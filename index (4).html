🗺 카카오맵 API 설정 가이드

1

카카오 개발자 센터 접속

developers.kakao.com → 카카오 계정 로그인 → 내 애플리케이션

2

JavaScript 키 복사

기존에 만든 앱(안심전세체크) → 앱 키 → JavaScript 키 복사
또는 새 앱 생성 후 키 복사

3

플랫폼 도메인 추가

앱 설정 → 플랫폼 → Web → 사이트 도메인:

https://mrlee0453-netizen.github.io
4

index.html에서 키 교체

파일 맨 아래 스크립트에서 이 줄을 찾아 교체:

const KAKAO_KEY = 'YOUR_KAKAO_API_KEY'; // ← 여기
5

GitHub에 파일 업로드

github.com → realprice-map 레포지토리 생성 → index.html 업로드
Settings → Pages → main branch → Save
